# 开发智能体 - 配置与编排

_Source: https://developer.huawei.com/consumer/cn/doc/service/developing-intelligent-agents-0000002435989592_

子章节：智能体分类、配置-基础信息、配置-隐私协议服务、配置-内容合规、编排-模型选择&角色指令、编排-能力拓展、调试与预览、上/下架升级流程介绍、智能体版本管理。

## 配置-基础信息

创建智能体后，可编辑智能体图标、名称、一句话描述、支持的设备和系统、创建者昵称、智能体分类、智能体别名、AgentCard。

- **图标**：建议1:1正方形图片，大小不超过5M，支持png、jpeg、jpg不透明背景
- **智能体名称**：必填项，不能编辑为空
- **智能体分类**：根据能力选择适合的分类
- **支持的设备和系统**：多选（HarmonyOS NEXT = ROM 5.X以上，HarmonyOS = ROM 4.X）
  - 手机/平板/PC/手表 - HarmonyOS NEXT
  - 手机/平板 - HarmonyOS
  - 车机 - HarmonyOS
- **创建者昵称**：不配置默认为账号设置的创建者昵称；企业开发者也可选择已创建并通过审核的创建者昵称
- **一句话描述**：显示在智能体详情页
- **智能体别名**：用于智能体分发，提高分发准确率，支持配置多个
- **AgentCard**：智能体的"名片"，描述能力和技能，主要用于小艺主对话分发智能体。根据基础信息与插件/工作流配置自动同步生成；可新增/编辑/删除智能体功能，可编辑功能名称（同步自插件工具名/工作流名）、功能描述、依赖项（同步自能力依赖的端插件，云插件无依赖项）。不可编辑字段：名称、ID、版本、描述、输入类型（同步自输入文件设置，可含字符串/文件/图片）、输出类型（预留，默认字符串）

## 配置-隐私协议服务

- **自定义隐私政策**：已有隐私网址时直接填写
- **隐私托管**：平台基于标准化模板生成隐私协议，包含：
  - 导语部分（智能体介绍、重要提示或版本变更说明）
  - 我们如何收集和使用您的个人信息（默认收集用户对话信息；其他数据项通过"新增数据信息"添加，填写采集用途、前提条件、补充说明；个性化推荐说明可选）
  - 对未成年人的保护（可选，填写儿童隐私保护声明链接）
  - 与第三方共享（可选，按公司声明）/ 第三方MCP Server（可选，按MCP Server声明）
  - 管理您的个人信息（固定内容）
  - 信息存储地点及期限（固定存储期限或"实现处理目的所必要的最短时间"；存储位置固定为中国境内服务器）
  - 开发者自定义章节（可选）
  - 如何联系我们（电话、邮箱、公司地址）
  - 隐私政策生效日期
- 每个智能体可配置多个隐私协议，但上架时仅能关联1个；需点击【生成协议】使协议变为"完成"状态后才可关联；编辑后需再次生成协议
- 平台会自动检测智能体编排涉及的隐私相关项与所选协议是否一致，不一致时提示开发者补充
- 上架后用户可在手机端智能体详情页-隐私和安全-隐私声明中查看

## 配置-内容合规

依据《生成式人工智能服务管理暂行办法》《互联网信息服务深度合成管理规定》《人工智能生成合成内容标识办法》等，上架前需填写：

- **人工智能生成合成内容标识**：是否包含AI生成合成内容文件（文本/图片/音频/视频/虚拟场景等）。选择"是"时勾选AI生成文件类型并填写标识元数据：
  - 生成合成标签 Label（1=属于、2=可能、3=疑似AI生成合成内容）
  - 生成合成服务提供者 ContentProducer、内容制作编号 ProducerID、预留字段1 ReservedCode1
  - 内容传播服务提供者 ContentPropagator、内容传播编号 PropagatorID、预留字段2 ReservedCode2
- **大模型备案信息**：接入非平台提供的三方大模型时需申报备案号，多个备案号用英文逗号分隔（生成式AI服务上线备案号、算法备案号）

## 编排-模型选择&角色指令

### 模型选择（单Agent/LLM模式）
- 支持选择模型，支持多模型对比调试
- 模型参数设置：携带上下文轮数（0-20轮）；部分模型支持超参：Top-K（1-256，单步1）、Top-P（0-1，单步0.01）、Temperature（0-1，单步0.01）

### 对话设置（工作流模式）
- 携带上下文轮数（Context Window）：最小0轮，当前开放最大20轮

### 角色指令（Prompt）
- 提示词对比调试：对比不同提示词的响应效果，选择最优
- 两种提示词优化方式：
  1. **数据集优化提示词**（多轮优化策略）：配置优化任务名称、待优化提示词、数据集、优化模型、算法评估模型、评估方法（模型自主评估=适用文案创作等非精确场景 / 文本精准评估=字符串对比，适用文本分类等精确场景）、生成提示词中的示例数量（最大10个）、优化轮次（最大20轮）
  2. **全文自动优化/根据调试结果优化**

## 编排-能力拓展

### A2A基础配置（A2A模式必须配置，否则无法发布）
- **API URL**：与智能体对话时的访问接口（鸿蒙Agent通信协议）
- **客户端服务器间会话维持方式**：由服务器侧分配Session；或服务器间无状态通信，每次携带认证凭据
- **认证信息**：AK/SK认证（预共享密钥）、OAuth认证（OAuth2.0，仅支持Client模式）、Header认证、Query认证（存在安全风险，建议用其他方式）

### A2A输出设置
- 用于绑定卡片输出，最多20条；需先完成A2A基础配置
- 每条配置：卡片名称、卡片描述、输出参数（默认必须包含参数名为`cardData`、类型为Object的参数，且必须添加子项）；添加输出后可绑定卡片

### OpenClaw基础配置（OpenClaw模式）
- 每个账号下仅限创建一个OpenClaw模式智能体；创建时设备默认勾选"手机-HarmonyOS NEXT"和"手机-HarmonyOS"
- 配置步骤：(1) 获取小艺开放平台凭证：【工作空间】-【凭证】新建凭证，生成Key和安全密钥（安全密钥仅创建时可明文复制）；(2) 在OpenClaw服务器上配置channel，将ak/sk替换为凭证的Key和安全密钥，其他配置不可更改
- 发布：仅支持发布真机测试供白名单用户体验（开发测试态长期有效），白名单人员可在小艺App对话页触达

### 开场对话&预置问题
- 智能体开场语：支持MarkDown输入（标题、加粗、斜体、列表、引用、链接等）
- 开场语预置问题：最多3个

### 输入文件设置
- 照片：支持上传（JPEG/JPG/PNG/BMP/WEBP）、支持拍摄（不支持网页调测，需真机体验）；发送方式：直接发送 / 进待发送区发送（可同时发文本+照片，单次最多6张）；支持照片快捷指令配置
- 文件：支持上传（pdf/doc/docx/txt/ppt/pptx/xlsx/xls）；发送方式同上（待发送区单次最多6个）；支持文件快捷指令配置

### 用户问题建议（LLM/工作流模式）
- 开关默认关闭；打开后每次智能体回复后为用户提供问题建议；可选【系统配置Prompt】或【自定义Prompt】

### 快捷指令
- 每个智能体最多10个静态快捷指令 + 1个动态快捷指令；支持拖动调整顺序，动态指令固定排最前（动态指令不足5条时按顺序取后面静态指令）
- 配置项：快捷指令文案、指令图标（可选）、位置固定底部（输入框上方）
- **类型**：
  - 跳转型（DEEP LINK）：应用名称、应用包名、DeepLink、app最小版本号、Action名称（可选）；暂不支持页面调试预览
  - 指令发送型：文档 / 图片 / 文本 / 拍摄（前置/后置镜头，可选拍后裁剪；不支持页面预览）/ 选择面板（组件：上传对象最多1个、图片模板最多1个、平铺选择器（单选/多选）最多2个、下拉选择器最多2个；系统自动组装"指令文本内容"+各组件信息生成query）
  - 状态型：用户可开关的功能。配置状态值（不能与关联变量默认值相同）、是否默认选中；选中状态下用户query携带该状态值；一个快捷指令只能关联一个变量，默认值为空的变量不能被关联
- **动态快捷指令**：通过执行配置的工作流动态生成。工作流模式下必须绑定主工作流。工作流开始节点需包含系统参数`EVENT_INPUT`（含`header.name`事件名），结束节点需输出`dynamicDirective[]`（每项含`content`显示内容、`query`发送给智能体的请求）和`eventInfo`（`directiveName: "QuickChips"`固定值、`eventName`）。事件：`HalfScreen`半屏态 / `FullScreen`全屏态进入时触发

### 背景图片
- 为智能体在小艺应用中增加对话背景图片

### 角色声音
- 试听开放音色并设置默认音色；【终端用户可以切换声音】开关默认开启

### 语音通话（暂不支持）
- 文档中标注"暂不支持"。包含支持外屏语音通话开关、支持后台运行开关和跳转页配置

### 插件
插件是工具集，一个插件可包含一个或多个工具（API）。智能体内插件配置子项：
- **插件参数设置**：执行设置（云/MCP插件：【是否包含异步任务】开关，开启后小艺APP不在前台时插件执行完自动发Push通知，默认关闭；端插件：执行类型前台/后台、执行方式立即执行/打断播报）；输入参数设置（默认值支持输入/引用变量；【是否启用】控制参数对大模型是否可见）；输出参数设置（【融合生成】开关控制大模型是否基于插件执行结果融合生成）
- **插件绑卡**：同一插件可绑定多张卡片，支持同一时刻输出多张卡片，可配置答复文本和卡片的输出顺序。批式插件文本与卡片可随意编排（融合生成关闭时只出卡）；流式插件支持在首帧（先卡后文）或末帧（先文后卡）返回绑卡信息。绑卡标识（仅String/Array[String]）可设置出卡条件，不设置则固定出卡
- **插件调试**：网页调试展示云插件的插件名/工具名/请求参数/结果；端插件依赖应用，需真机体验
- **插件升级**：云插件版本更新后再次进入智能体自动更新（仅测试态；上架版本需升级智能体）；端插件需手动移除旧版本并添加新版本
- **插件模拟集**：打开MockSet开关后插件响应模拟数据，无需真实接口；仅网页调试生效，手机端调用走真实接口

### 工作流/工作流配置
工作流用于处理功能类请求，通过顺序执行节点实现功能。子项：
- **工作流执行设置**：【是否包含异步任务】开关，开启后工作流执行完成且小艺APP不在前台时自动发Push消息通知结果（默认关闭）
- **工作流绑卡**：支持为输出节点和结束节点绑卡，单节点支持多张卡片、可配置文本与卡片输出顺序。非流式输出节点绑卡只出卡不出文本（需同时输出可用两个输出节点）；流式输出节点可同时输出卡片和文本；思考模式的输出节点不支持绑卡。流式结束节点支持首帧/末帧返回绑卡数据；返回变量的结束节点无答复文本只输出卡片。绑卡标识规则同插件绑卡
- **工作流文卡混排输出**：答复数据源必须来自**流处理云插件**；插件出卡帧`items.displayType`需为`EmbedMarkdown`；输出/结束节点绑卡变量名必须为`items`且开启"流式输出"开关；插件必须与答复节点直接相连（中间不得插入其他节点或并联其他流式输出节点）
- **工作流界面跳转配置**：以界面形式返回结果。先开发界面（预留界面入参变量），工作流在绑定界面的节点输出界面所需变量；智能体中点击【绑定卡片或界面】-界面跳转-添加已发布界面，再做界面入参配置（与输出/结束节点变量绑定映射）。界面功能暂不支持网页端调试，需真机测试

### 触发器
使智能体在接收到特定事件时自动执行任务。支持**通用事件**和**Webhook事件**。
- **通用事件**：半屏态或全屏态进入智能体时触发。仅LLM模式和A2A模式支持。一个通用事件仅可关联一个触发器。任务执行类型：智能体提示（预设内容发送给智能体）、插件、工作流（均支持"是否保存会话上下文"开关）。A2A模式不支持任务执行配置——触发时把绑定的通用事件及参数通过A2A基础配置的API URL传给三方智能体处理。注意：触发器触发工作流时无USER_INPUT，需在开始节点定制其他输入参数；绑定的插件/工作流升级后需手动重新绑定
- **Webhook事件**：异步事件场景（内容生成/任务执行完成后通知用户）。开发者向华为统一接口`https://hag.cloud.huawei.com/open-ability-agent/v1/agent-webhook`发送通知消息；用户手机端收到通知，点击进入智能体查看结果。消息类型：text（直接展示）或data（作为入参触发任务执行，结果答复用户）；同时发送时只有text生效。配置字段：事件选项Webhook、URL、Access Key & Security Key（凭证密钥对）、apiID（配置确认后查看）、任务执行（插件 / 工作流 / 事件结果按A2A协议返回给智能体处理）。请求头：`x-hag-trace-id`、`X-Access-Key`、`X-Sign`（Base64(HMAC-SHA256(secretKey, ts))）、`X-Ts`（毫秒时间戳）；请求体为JSON-RPC 2.0，含`result.apiId`、`result.pushId`（系统变量push_id）、`result.agentLoginSessionId`（账号绑定智能体必填）、`result.pushText`（最多84英文字符/57中文字，最多3行）、`result.kind: "task"`、`result.artifacts[]`（parts的text/data二选一）。工作流任务的data参数会塞入工作流入参`EVENT_INPUT.payload`

### 关联应用
智能体与应用关联，支持在应用中一键呼出半屏态智能体。
- 前提条件：已在AppGallery Connect创建鸿蒙应用、已配置**手动签名**（自动签名会导致appid随机、授权异常）、平台已创建智能体并关联应用（仅支持本账号在AGC已上架的应用）、应用已集成AgentKit能力（通过Function组件拉起智能体）
- 调试要求：设备登录华为账号并联网、智能体已发真机测试；ROM 6.0.0+、API SDK 20+、小艺APP 11.3.8.300+
- 常见问题："智能体未授权给该应用"=未配置关联/名称包名appid不一致/自动签名；无组件图标=未集成AgentKit或ROM/SDK版本低；上架后仍呼出"开发中"智能体=真机白名单设备优先拉起开发版，删除白名单或取消真机发布解决

### 账号绑定设置（仅A2A模式和工作流模式）
- **A2A模式**：打开账号绑定开关并输入APP ID（绑定应用的Client ID），保存后联系华为侧支撑人员走内部评审。用户绑定账号时请求A2A基础配置中的地址获取会话凭证
- **工作流模式**：APP ID + 账号授权服务设置（API URL：绑定账号时调用此地址获取登录凭证；认证信息：AK/SK、Header、Query）。配置完成后自动关联系统变量`agent_login_session_id`（取授权接口响应中的agentLoginSessionId），工作流中对接开发者接口的节点（如云插件）引用该变量判断登录状态
- 用户绑定方式：
  1. superlink引导登录（适用强制绑定场景）：`[同意并绑定账号>>](superlink://vassistant?hwIdAuth=phone&appId={{APP_ID}}&agentId={{agentId}})`
  2. 智能体详情页点击绑定手机号栏进行绑定/解绑
- 开发准备：在AGC创建项目和web应用获取APP ID（取应用信息中的Client ID）；产品名称和应用图标需与智能体名称、图标一致。"获取您的手机号"权限**仅企业开发者**可申请（Phone、Tablet设备），通过联盟"管理中心-授权管理-华为帐号服务"提交敏感权限申请及场景材料

### 付费智能体
智能体内提供付费数字商品或服务（虚拟点卡、游戏道具、游戏货币、会员权益等）时，可集成"智能体数字商品支付服务"。目前**内测阶段，仅企业开发者可用**，需通过商务或华为开发者联盟工单联系小艺商业化运营人员申请开通。详见智能体数字商品支付服务配置以及使用指导。

### 知识库（LLM模式）
- 【检索文本相关阈值】：召回段落的相关系数，范围0-1（越高返回片段完整度/相关度越高）
- 【最大召回分段数】：范围0-20
- 【最大召回token数】：范围0-999999
- 【查询改写】：根据对话历史优化/重构用户Query，默认开启
- 【回复】（无召回回复）：默认（"抱歉，这个问题不在知识范围内"）或自定义

### 变量
- **用户变量**：持久化存储每用户动态数据（语言偏好、个性化设置等），可勾选是否在角色指令、工作流中使用。抽取规则：工作流模式不支持基于上下文抽取（只能在工作流变量组件中更新）；LLM模式未绑定工作流时支持基于上下文抽取更新；绑定工作流时，执行流程含变量组件则不支持抽取，不含则支持
- **系统变量**：用户使用智能体时自动产生的数据，只读，不可新增/修改/删除；可在工作流和插件中使用；开关默认关闭。平台提供设备标识符、设备语言等
- 查看/重置：右上角【记忆】-【变量】

### 长期记忆（LLM/工作流模式）
- 提取多轮对话中的内容并保存，用于个性化回复。上下文轮数决定抽取时机（如设20轮，则第21轮对话后从前20轮提取）
- 默认【支持在Prompt中调用】（后台自动拼接，无需显式调用）；取消勾选后仅能在Workflow中调用
- 查看/重置：右上角【记忆】-【长期记忆】

## 上/下架、升级流程

- 上架并审核通过后才能被其他用户搜索使用；升级审核未通过时，线上保留上一次通过审核的版本；下架后用户无法使用，可编辑后再次提交上架
- **当前暂不支持上架包含私有云插件及私有MCP插件的个人智能体**（仅支持页面调试）
- 上架前需通过平台校验（检查清单）
- 团队账号下仅团队管理员可发起上下架等操作
- 操作入口：编排页右上角【上架】/【升级】，或【工作空间】-【智能体】列表操作区（上架/下架/升级/撤回）
- 审核周期1-3个工作日
- 审核不通过原因：编排页【版本记录】查看，或智能体列表状态tag悬浮查看

## 智能体版本管理

- 版本记录包括智能体上下架及撤回审核时的版本；编排页右上角【版本记录】查看历史版本（查看时不可编辑）
- 回退到历史版本：将历史版本覆盖当前草稿态版本。回退原则：
  - 关联资源（插件/工作流/卡片/知识库）已更新版本 → 自动关联资源最新版本
  - 资源已下架或删除 → 自动删除关联关系
  - 除插件/工作流/知识库/卡片外，其他配置均可完全还原
