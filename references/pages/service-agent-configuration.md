# 开发智能体 - 配置与编排

_Source: https://developer.huawei.com/consumer/cn/doc/service/developing-intelligent-agents-0000002435989592_

## 配置-基础信息

创建智能体后，开发者可以编辑智能体图标、名称、一句话描述、支持的设备和系统、创建者昵称、智能体分类、智能体别名、AgentCard。

- **图标**：1:1正方形图片，大小不超过5M，支持png、jpeg、jpg不透明背景
- **智能体名称**：必填项，不能为空
- **智能体分类**：根据能力选择适合的分类
- **支持的设备和系统**：多选
  - 手机/平板/PC/手表 - HarmonyOS NEXT（ROM版本5.X以上）
  - 手机/平板 - HarmonyOS（ROM版本4.X）
  - 车机 - HarmonyOS
- **一句话描述**：显示在智能体详情页
- **智能体别名**：用于智能体分发，提高分发准确率
- **AgentCard**：自动同步自插件与工作流配置的功能列表，可编辑功能名称、描述、依赖项

## 配置-隐私协议服务

- **自定义隐私政策**：直接填写隐私网址
- **隐私托管**：使用平台模板生成隐私协议，包含：
  - 导语部分（智能体介绍、重要提示或版本变更说明）
  - 我们如何收集和使用您的个人信息
  - 对未成年人的保护（可选）
  - 与第三方共享 / 第三方MCP Server（可选）
  - 管理您的个人信息（固定内容）
  - 信息存储地点及期限（存储位置固定为中国境内）
  - 开发者自定义章节（可选）
  - 如何联系我们
  - 隐私政策生效日期
- 每个智能体可配置多个隐私协议，但上架时仅能关联1个

## 配置-内容合规

- **人工智能生成合成内容标识**：是否包含AI生成合成内容文件（文本/图片/音频/视频/虚拟场景等）
  - 选择"是"时需填写标识元数据：Label、ContentProducer、ProducerID、ReservedCode1、ContentPropagator、PropagatorID、ReservedCode2
- **大模型备案信息**：接入非平台提供三方大模型时需申报备案号

## 编排-模型选择&角色指令

### 模型选择（LLM模式）
- 支持多模型对比调试
- 超参设置：携带上下文轮数（0-20）、Top-K（1-256）、Top-P（0-1）、Temperature（0-1）

### 对话设置（工作流模式）
- 携带上下文轮数：最小0轮，最大20轮

### 角色指令（Prompt）
- 提示词对比调试
- 两种提示词优化方式：
  1. 数据集优化提示词：设置优化任务名称、数据集、优化策略、评估方法（模型自主评估/文本精准评估）、优化轮次（最大20轮）
  2. 全文自动优化/根据调试结果优化

## 编排-能力拓展

### A2A基础配置
- API URL：智能体对话时的访问接口
- 客户端服务器间会话维持方式：由服务器侧分配Session，服务器间采用无状态通信，每次携带认证凭据
- 认证信息：AK/SK认证、OAuth认证（仅支持Client模式）、Header认证、Query认证

### A2A输出设置
- 卡片名称、卡片描述
- 输出参数：默认包含参数名为cardData、类型为Object的参数
- 最多可添加20条输出配置

### OpenClaw基础配置
- 每个账号下仅限创建一个OpenClaw模式智能体
- 获取小艺开放平台凭证（Key和安全密钥），在OpenClaw服务器上配置channel（ak/sk替换）
- 仅支持发布真机测试供白名单用户体验

### 开场对话&预置问题
- 智能体开场语：支持MarkDown输入
- 开场语预置问题：最多3个预置问题

### 输入文件设置
- 照片：JPEG、JPG、PNG、BMP、WEBP；支持拍摄；发送方式：直接发送/进待发送区发送（最多6张）
- 文件：pdf、doc、docx、txt、ppt、pptx、xlsx、xls；发送方式：直接发送/进待发送区发送（最多6个）

### 快捷指令
- 最多10个静态快捷指令 + 1个动态快捷指令
- 类型：
  - 跳转型：DEEP LINK（应用名称、包名、DeepLink、最小版本号、Action名称）
  - 指令发送型：文档/图片/文本/拍摄/选择面板（上传对象/图片模板/平铺选择器/下拉选择器）
  - 状态型：关联变量，开关切换
- 动态快捷指令：工作流驱动的场景化快捷指令，半屏态/全屏态事件触发

### 角色声音
- 试听开放音色，为智能体设置默认音色
- 终端用户可以切换声音开关（默认开启）

### 触发器
- 通用事件和Webhook事件

### 关联应用
- 智能体与应用关联，支持在应用中一键呼出智能体
- 前提条件：已在AppGallery Connect创建鸿蒙应用、已配置签名、应用APP端已集成AgentKit能力
- 调试要求：ROM 6.0.0+、API SDK 20+、小艺APP 11.3.8.300+

### 账号绑定设置
- 仅A2A模式和工作流模式支持
- A2A模式：打开账号绑定开关并输入APP ID（Client ID），联系华为侧支撑人员走内部评审
- 工作流模式：配置APP ID + 账号授权服务API URL + 认证信息
- 自动关联系统变量agent_login_session_id
- 用户绑定方式：
  1. superlink引导登录：`superlink://vassistant?hwIdAuth=phone&appId={{APP_ID}}&agentId={{agentId}}`
  2. 智能体详情页点击绑定手机号栏

### 知识库
- 检索文本相关阈值（0-1）、最大召回分段数（0-20）、最大召回token数（0-999999）
- 查询改写：默认开启
- 无召回回复：默认（"抱歉，这个问题不在知识范围内"）或自定义

### 变量
- 用户变量：持久化存储每用户动态数据，可在角色指令和工作流中使用
- 系统变量：自动产生的数据，只读，如设备标识符、设备语言等

### 长期记忆
- 多轮对话中提取并保存内容
- 默认支持在Prompt中调用，取消勾选后仅能在Workflow中调用
- 查看见右上角【记忆】-【长期记忆】

## 上下架及版本管理

- 审核周期：1-3个工作日
- 团队账号下仅团队管理员可发起上下架操作
- 升级审核未通过时，线上保留上一次通过审核的版本
- 版本记录中可查看历史版本，回退规则：
  - 资源已更新版本→自动关联最新版本
  - 资源已下架或删除→自动删除关联关系
  - 插件/工作流/知识库/卡片外，其他配置均可完全还原
