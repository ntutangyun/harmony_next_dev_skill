# 跨设备协同通知概述

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/notification-distributed-overview_

跨设备协同支持的设备：从API version 18开始，支持手机与手表之间通知消息的协同；从API version 20开始，支持手机与平板、2in1设备之间通知消息的协同。
跨设备协同支持的通知渠道：
手表：带快捷回复的社交通信类通知（社交通信）、实况窗。
平板：社交通信、服务提醒、实况窗、客服消息。
2in1：社交通信、服务提醒、客服消息。
