# @copyright

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-arktsdoc-copyright_

@copyright标签用于记录文件的版权信息。

语法

@copyright <some copyright text>

示例

/**
 * @copyright Michael 2025
 */
export function myFunction() {}
