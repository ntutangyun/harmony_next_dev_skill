# 开通云函数服务

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cloudfoundation-enable-function_

首次使用云函数服务前，需要先开通此服务。如果已经启用，可跳过本步骤。

登录AppGallery Connect，点击“开发与服务”。

在项目列表中点击需要开通云函数的项目。

在左侧导航栏选择“云开发（Serverless）> 云函数”，进入云函数页面，点击“立即开通”。

说明

如果开发者此时未设置数据处理位置，系统会自动弹出提示框引导开发者完成设置。目前，云函数支持启用多个数据处理位置，具体请参见设置数据处理位置。

如果开发者已启用多个数据处理位置，当需要在不同的数据处理位置管理云函数时，可在云函数页面选择“数据处理位置”下拉选项进行切换。
