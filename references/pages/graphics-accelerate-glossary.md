# Graphics Accelerate Kit术语

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/graphics-accelerate-glossary_

B

[h2]Background Asset Download；资源包后台下载

游戏资源包的后台静默下载服务，支持在游戏安装后、强制更新版本后或设备闲时自动触发游戏的资源包后台下载。

C

[h2]Content Delivery Network（CDN）

内容分发网络（Content Delivery Network，简称CDN），是一种通过网络中分布资源服务器，用以提高网站访问速度的技术。当前资源包后台下载功能支持游戏在AppGallery Connect配置华为CDN或三方CDN的资源包下载配置项。

D

[h2]Deep Freeze；深度冻结

游戏进程退出时系统将进程冻结，取消CPU（Central Processing Unit）对其调度，异步IPC（Inter-Process Communication）无法唤醒。

M

[h2]Memory Mirror；内存镜像

游戏上划退出时系统将90%以上的内存换出（Swap Out）至磁盘，内存中仅保留少量关键对象。

O

[h2]Open GPU Turbo X（OpenGTX）

Graphics Accelerate Kit中面向游戏场景的开放接口，用于动态调控LTPO（Low Temperature Polycrystalline Oxide）屏幕刷新率、GPU（Graphics Processing Unit）、DDR（Double Data Rate）频率与渲染负载，以在高帧率重载游戏中平衡流畅度与功耗。‌‌‌

P

[h2]Pre-launch；预启动

游戏启动加速服务根据用户历史使用习惯，提前加载资源和游戏部分初始化。
