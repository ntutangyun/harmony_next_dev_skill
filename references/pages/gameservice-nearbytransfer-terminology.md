# 游戏近场快传术语

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/gameservice-nearbytransfer-terminology_

I

[h2]Installation Package Transfer；安装包传输

当玩家中有未安装游戏的玩家时，已安装游戏的玩家将游戏安装包通过碰一碰或隔空传送传输给未安装游戏的玩家的能力，实现游戏传播效率的提升。

[h2]Integration-free Transfer；免集成传输

近场快传场景下支持的简化安装包传输流程的能力，开发者无需额外集成即可实现安装包的近场传输。

N

[h2]Nearby Transfer；游戏近场快传

设备在靠近的情况下进行游戏数据传输，支持传输资源包和安装包两种游戏数据。

P

[h2]Package Information；游戏包信息

标识游戏应用的基本信息，包括包名、应用版本等。

R

[h2]Resource Package Transfer；资源包传输

已更新游戏版本的玩家将自己设备上的游戏资源包发送给未更新玩家的能力。可节省玩家下载流量和时间，降低因频繁更新导致用户流失的可能性。
