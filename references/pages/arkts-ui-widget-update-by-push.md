# ArkTS卡片Push刷新

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-ui-widget-update-by-push_

卡片Push刷新是一种通过推送服务（Push Kit）远程刷新服务卡片数据的机制。卡片Push刷新允许应用服务端通过推送服务（Push Kit）向用户设备发送消息，直接更新卡片上的内容，而无需用户主动打开应用或依赖周期性刷新。卡片Push刷新流程请参考推送卡片刷新消息。

约束与限制

根据应用所属分类，推送服务对每日推送频次设有上限（详见下表）。若推送请求超过对应分类的上限，系统将自动拦截并拒绝推送。

影视直播	影视与直播功能卡片	1.平台信息提示 2.关注主播直播开始提示	30
