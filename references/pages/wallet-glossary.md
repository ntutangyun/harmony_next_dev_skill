# Wallet Kit术语

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/wallet-glossary_

A

[h2]Applet

在安全单元中实现相关安全功能的应用，数字车钥匙的加密、签名、密钥存储等核心安全功能均在此Applet中实现。

D

[h2]Digital Key (DK); 数字钥匙

华为数字车钥匙的简称，通过移动终端设备实现车钥匙的功能。

I

[h2]Intelligent Car Key Ecosystem (ICCE); 智慧车联产业生态联盟数字车钥匙系统

由中国信通院及中国汽车工业协会牵头，华为、荣耀等手机厂商及相关车企共建的数字车钥匙行业标准，满足ICCE标准可快速实现移动终端模拟车钥匙的功能，实现碰一碰解锁和无感解锁功能。

S

[h2]SE (Secure Element); 安全单元

位于手机、穿戴设备中的一种防篡改硬件安全部件，用于保证内部应用和数据的安全性与机密性，在车钥匙场景典型安全单元为eSE。
