编辑.ohpmrc文件，或使用命令ohpm config set <key> <value>设置仓库地址（如ohpm config set registry https://ohpm.openharmony.cn/ohpm/）。

00638001 命令行中使用了无效选项

错误信息

Invalid Option.

错误描述

命令行中使用了无效的选项。

可能原因

在命令行中输入的选项无效，可能是拼写错误、选项不支持或参数格式错误。

处理步骤

检查和确认命令支持的选项后再执行。
