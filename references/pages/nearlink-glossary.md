# NearLink Kit术语

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/nearlink-glossary_

A

[h2]Asynchronous Connection-Oriented Bidirectional (ACB)；面向连接的异步双向链路

星闪技术中的一种异步双向数据传输链路，用于在设备间可靠地传输数据。

C

[h2]Coordinated Devices Set Management (CDSM)；合作设备集合管理

对合作设备集合的管理。合作设备集合是由一组合作设备成员组成的、提供特定服务的统一整体。

S

[h2]SparkLink Service Access Protocol (SSAP)；星闪服务交互协议

定义了服务结构的发现过程和访问过程，以及过程中需要使用的信令。通过服务管理，星闪设备之间能够实现服务层面的互联互通。
