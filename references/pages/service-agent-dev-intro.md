# 鸿蒙智能体

_Source: https://developer.huawei.com/consumer/cn/doc/service/developer-guide-0000002469667881_

鸿蒙智能体是运行在HarmonyOS设备上的AI智能体，通过小艺开放平台开发、调试和分发。开发者可以创建智能体，配置其能力，然后发布到智能体市场供用户使用。文档"鸿蒙智能体"章节包含：快速创建智能体、开发智能体、开发知识库、开发工作流、开发插件、开发卡片、开发界面、开发音色、小艺罗盘（评测/观测）、上架审核规范。

## 智能体分类（编排模式）

_详见：https://developer.huawei.com/consumer/cn/doc/service/differences-in-arrangement-modes-0000002471344117_

### 单Agent（LLM模式）
基于大模型的智能体编排方式。开发者按需选择大模型，根据业务逻辑编写提示词，以LLM为理解中枢，结合意图识别、工具调用、对话上下文，动态选择插件、工作流，响应用户需求。适用于简单对话、知识问答、基础内容生成等场景。

### 工作流模式
基于规则化流程的智能体编排方式。开发者将复杂任务拆解为有序的规则化步骤（如数据获取、处理、执行），串联插件、大模型、条件分支、代码块等组件实现自动化执行流程。适用于需多步骤协同、逻辑复杂、业务多样性的场景。

### A2A模式
三方智能体接入小艺开放平台的高效编排方式。开发者可通过该模式基于鸿蒙Agent通信协议快速、便捷地将成熟的第三方智能体对接至小艺开放平台，实现分发与调用，提升平台的场景覆盖能力。该模式适用于同时具备鸿蒙端应用与云侧智能体能力的企业开发者。

### OpenClaw模式
开放灵活的智能体接入与构建方式，开发者可通过该模式接入OpenClaw工具，快速创建个性化智能体。该模式适用于个性化助手、自动化服务、场景化应用等多样化需求。（每个账号仅限创建一个OpenClaw模式智能体，仅支持发布真机测试供白名单用户体验。）

## 快速创建智能体

_详见：https://developer.huawei.com/consumer/cn/doc/service/quick-start-0000002469548009_

1. 登录小艺开放平台，进入小艺智能体平台页面，点击左上角【+创建智能体】。
2. 在标准创建页面填写智能体设定：名称、头像、智能体描述（直接显示在详情页）、智能体分类、支持的设备等，选择编排模式后点击【创建】进入编排页面。
3. 在编排页面为智能体添加能力（不同编排模式的能力拓展功能点有所区别）：开场对话（让用户快速了解功能或场景设定）、预置问题（点击即可快速体验）、角色指令prompt（直接决定智能体效果）。
4. 利用调试与预览工具体验实际效果、定位问题。
5. 点击右上角【保存】，再点击【上架】，等待审核通过后即可在端侧小艺发现页搜索到该智能体。优质智能体有机会优先展示，高价值智能体有机会通过端侧小艺发现页直接展现。
