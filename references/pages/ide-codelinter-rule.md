# Code Linter代码检查规则

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-codelinter-rule_

Code Linter代码检查规则的内容包括规则功能、配置选项、正/反示例代码等。

规则中的正/反示例代码主要用于帮助开发者直观地理解规则的检测位置，以及如何修改代码。由于运行环境等差异，直接执行正/反示例代码可能会出现运行失败或其它问题。若需要运行示例，开发者可能需要进一步自定义开发或修改，使示例代码完整可执行。
