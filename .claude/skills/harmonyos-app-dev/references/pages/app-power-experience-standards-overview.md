# 概述

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/app-power-experience-standards-overview_

前台场景：当应用或元服务处于设备屏幕的顶层时，用户可以直接看到并进行操作，即为前台场景。前台场景功耗规范提供了关于应用前台帧率、前台绘制渲染、应用前台资源使用的功耗规范。
后台场景：当应用或元服务未处于设备屏幕的顶层时，用户无法直接看到或进行操作，即为后台场景。后台场景功耗规范提供了关于后台任务、后台硬件资源、后台软件资源的使用规范。
