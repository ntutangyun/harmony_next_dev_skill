# 测试屏幕朗读功能

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/test-screen-reader_

下述列举了屏幕朗读常用的手势及对应的操作，更多操作手势可通过“设置 > 辅助功能 > 屏幕朗读 > 更多设置 > 自定义快捷手势”进行查看和设置。

表1 屏幕朗读基本手势

操作	手势
移至屏幕的下一项内容	向右滑动/向下滑动
移至屏幕的上一项内容	向左滑动/向上滑动
点击/选择所在项	点按两次
长按	点按两次后长按
横滑	双指滑动

表2 屏幕朗读往复手势

操作	手势
上一个导航设置	先向上再向下
下一个导航设置	先向下再向上
向下（后）滚动一个页面	先向右再向左
向上（前）滚动一个页面	先向左再向右

表3 屏幕朗读角度手势

操作	手势
主屏幕	先向上再向左
返回	先向下再向左
通知栏	先向上再向右
最近运行过的应用	先向下再向右
快捷菜单	先向右再向下
打开控制中心	先向左再向下
操作编辑框	先向左再向上
说明

为了更便捷地使用屏幕朗读，可通过“设置 -> 辅助功能 -> 屏幕朗读 -> 更多设置”进行更多选项（如语速、语调等）的设置。

使用屏幕朗读浏览应用

在完成屏幕朗读与应用的适配后，可参照以下步骤来验证朗读功能是否正确实现。

打开应用。

按顺序滑动浏览各个元素。

在浏览过程中，可注意以下几个问题：

每个元素的语音反馈是否恰当地传达了其内容或用途？

您能否轻松完成主要工作流程？

您能否通过滑动浏览每个元素？

如果出现提醒或其他临时消息，屏幕朗读能否读出来？

参考开发指南解决适配导致的问题。
