# 个人数据处理说明

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/scenario-fusion-personal-data_

在表单中填写过的信息保存至历史表单后，可前往“设置 > 隐私和安全 > 智能填充 > 历史表单输入”删除保存的历史表单输入记录。
可前往“设置”>“隐私和安全”>“智能填充”关闭“华为账号”开关，在“智能填充”缓存的华为账号相关数据会自动删除。

指导开发者如何帮助最终用户实现对数据的控制

开发者通过智能填充服务获取的用户数据，需要开发者自行提供对应的数据主体权利。
智能填充服务存储的用户个人数据请参考《关于智能填充与隐私的声明》。
