# ArkTS卡片界面开发概述

_Source: https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-ui-widget-page-overview_

ArkTS卡片具备JS卡片的全量能力，并且新增了动效能力和自定义绘制的能力，支持ArkTS声明式开发范式的部分组件、事件、动效、数据管理、状态管理能力。

对于支持在ArkTS卡片UI界面中使用的接口，会添加“卡片能力”的标记，如：从API version 12开始，该接口支持在ArkTS卡片中使用。同时请留意卡片场景下的能力差异说明。

例如：以下说明表示CircleShape可在ArkTS卡片中使用。
